# TDP-43 Network Perturbation Analysis in C9-FTD/ALS

This repository contains the code used to investigate TDP-43 protein interaction perturbations in C9orf72-associated Frontotemporal Dementia and Amyotrophic Lateral Sclerosis (c9-FTD/ALS). Using the UniPPIN protein-protein interaction network (Chung et al., 2018), two networks were constructed: a regulatory network representing normal TDP-43 interactions, and a perturbed network simulating the depletion of 119 key TDP-43 interactors sequestered by arginine-rich dipeptide-repeat proteins in c9-FTD/ALS (Boeynaems et al., 2017). Five random control networks were generated by removing an equivalent number of non-perturbed genes to validate that observed effects were specific to TDP-43 interactor depletion. Community detection was performed on all networks using the M1 algorithm from the MONET toolbox (Tomasoni et al., 2020) with a largest-module parameter of 80. Gene Ontology enrichment analysis of the resulting TDP-43-containing modules revealed a functional shift from nuclear RNA-binding and mRNA processing in the regulatory network, to ribosomal assembly and cytoplasmic translation in the perturbed network, supporting the hypothesis that TDP-43 mislocalisation is driven by the depletion of its key RNA-binding interactors.
